### 1. 기본 프롬프트
```text
1. 프로젝트 개요
초등학교 교실에서 선생님이 별도 설치 없이 즉시 사용할 수 있는 "폭탄 돌리기" 웹 앱을 만들어줘.
- 파일 1개(index.html)만 생성
- font는 jua를 사용
- HTML + CSS + JavaScript를 모두 이 파일 하나에 포함
- 브라우저에서 바로 실행되도록 외부 서버나 패키지는 사용하지 않음
- 허용되는 외부 리소스: Google Fonts CDN <link>태그 1개만

2. 화면 구성 (3단계 전환)
1단계: 설정 화면
- 최소 시간(초), 최대 시간(초) 숫자 입력 필드(기본값: 최소 10초, 최대 30초)
- 유효성 검사: 두 값 모두 1 이상의 정수, 최소 < 최대(위반 시 귀여운 경고 문구 표시)
- "시작!" 버튼 클릭 시 2단계로 전환
- 남은 시간을 최소 시간과 최대 시간 사이에서 램덤하게 설정하고 2단계로 전환
2단계: 폭탄 동작 화면
- SVG 폭탄 캐릭터를 중앙에 크게 표시
- 남은 시간이 줄어듬. 남은 시간은 표시 금지. 남은 시간 종료 시 3단계로 전환
- "그만하기" 버튼: 클릭 시 1단계로 돌아감
- 시각 피드백:
폭탄이 불규칙한 타이밍으로 갑자기 크게 흔들렸다가 잠잠해지는 패턴 반복
도화선 끝에 있는 불꽃의 크기가 랜덤하게 커졌다 작아졌다 변함
가끔 폭탄 눈이 깜짝 놀란 표정으로 바뀌었다가 돌아오는 연출
폭탄 아래에 언제 터질지 모르는 긴장감이 느껴지는 문장을 반복적으로 보여줌
- 청각 피드백:
Web Audio API를 이용해 시계음을 반복적으로 재생하여 긴장감을 높임
3단계: 폭발 화면
화면 전체를 덮는 폭발 효과 + 흔들림 애니메이션
정중앙에 "펑!" 텍스트를 크게 표시
Web Audio API로 폭발음 재생
3초 후 자동으로 1단계로 복귀

3. 그래픽 명세
모든 그래픽은 외부 이미지 파일 없이 인라인 SVG로 직접 작성
- 폭탄 SVG
둥글고 귀여운 검정 몸체에 광택 하이라이트 표현
눈 두 개(흰 원 + 검정 동공 + 흰 반짝이)와 살짝 웃는 입 (평상시 표정)
가끔 눈이 커지고 입이 벌어지는 놀란 표정으로 전환됐다가 복귀
폭탄 상단에서 시작해 오른쪽 위로 휘어진 선으로 도화선 표현
불꽃의 중심을 도화선 끝점에 일치하여 배치
불꽃은 노랑, 주황 색상으로 이루어진 방사형으로 뻗는 뾰족한 광선 8~12개
불꽃은 랜덤한 속도와 크기로 변함
- 폭발 SVG
중심에서 방사형으로 뻗는 뾰족한 광선 8~12개
노랑, 주황, 빨강 색상을 섞어 역동적으로 표현
등장 시 튀어나오는 팝 애니메이션
- 배경 장식 SVG
별, 하트, 원 모양을 인라인 SVG로 배경에 10~15개 흩뿌림
배경 장식이 비가 오듯이 반복적으로 흔들리며 떨어짐

4. 오디오 명세
모든 오디오는 외부 오디오 파일 없이 Web Auidio API로 직접 작성
- 시계음 리듬
'틱'과 '톡'이 번갈아 나타나는 2박자 리듬을 구현
틱: 높은 음의 화이트 노이즈. 하이패스 필터를 사용하여 날카롭고 짧게 '착' 하는 소리
톡: '틱'보다 약간 낮은 음. 로우패스 필터를 살짝 섞어 묵직하고 둔탁하게 '툭' 하는 소리
- 폭발음
저음의 진동과 넓은 대역의 노이즈를 합성하여 '콰광' 하는 입체적인 소리
```

### 2. 응용 추가 프롬프트
```text
[추가 프롬프트]
폭탄을 여러 개 설정할 수 있도록 다음과 같이 수정해줘.

추가 사항
1단계: 설정 화면에 요소 추가
기존 항목(최소 시간, 최대 시간) 유지하고, 아래 항목 추가
폭탄 개수 입력 필드 (기본값: 4개, 최소 1개 최대 10개)
유효성 검사: 1~10 사이의 정수만 허용, 위반 시 귀여운 경고 문구 표시
2단계: 폭탄 동작 화면 변경
설정한 폭탄 개수만큼 SVG 폭탄 캐릭터를 화면에 배치
각 폭탄은 독립적으로 최소~최대 시간 사이의 랜덤 시간을 따로 부여받음
폭탄마다 모둠 번호(1모둠, 2모둠 … )를 이름표처럼 표시
각 폭탄은 자신의 남은 시간이 종료되면 개별적으로 폭발(다른 폭탄에 영향 없음)
폭발한 폭탄은 제자리에서 SVG 폭발 이펙트 + "펑!" 텍스트로 전환되고, 해당 자리에 고정
모든 폭탄이 터진 후 일정 시간이 지나면 자동으로 1단계로 복귀
```
